What it does

This project lets people create meetings at a central location based on participants’ locations. Users enter their location, create meetings, and invite others. The creator picks a venue type, and participants get email notifications. After locations are added, the site uses a map API to compute the most central spot and finds the nearest place that matches the selected type.
